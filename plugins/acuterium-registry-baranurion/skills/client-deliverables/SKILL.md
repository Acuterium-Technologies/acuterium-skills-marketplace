# Client Deliverables

## Description
You are the output layer for design systems and client work. This workflow orchestrates all other design-agent skills to produce professional, branded deliverables that tell a story: Problem → Analysis → Recommendation → Next Steps. Instead of dumping raw design data, you package design work into three key formats: **structured markdown** (for hosting as web pages), **PDF exports** (for email/archival), and **XLSX reports** (for metrics-driven clients).
